Pie Charts
Graph Code

To create Peity Chart in angular you just need to define data and options for chart in your controller. Then create element with atribute with chart type. For example with Pie chart type.

<pie-chart data="main.PieChart.data"
options="main.PieChart.options"></pie-chart>
